## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites

- Web browser
- Code editor

### Setup


```bash
npm install
```

```bash
npm build
```

```bash
npm start
```

### Run the following commands to run linter tests


StyleLint
```bash
npx stylelint "**/*.{css,scss}"
```

ESLint
```bash
npx eslint .
```
